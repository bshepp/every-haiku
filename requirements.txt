# Every Haiku Project Requirements

## Development Tools
- Node.js >= 20.0.0
- npm >= 10.0.0
- Firebase CLI >= 13.0.0

## Firebase Services
- Firebase Authentication
- Cloud Firestore
- Firebase Hosting
- Firebase Cloud Functions (v2 API)
- Firebase Emulator Suite (for local development)

## Frontend Dependencies
- TailwindCSS (via CDN)
- Firebase JavaScript SDK v11.10.0 (via CDN, compat mode)

## Backend Dependencies (Cloud Functions)
- firebase-admin: ^13.4.0
- firebase-functions: ^6.4.0
- node-fetch: ^2.7.0

## Development Dependencies
- eslint: ^8.15.0
- eslint-config-google: ^0.14.0
- firebase-functions-test: ^3.3.0
- jest: ^29.7.0
- @firebase/rules-unit-testing: ^3.0.1
- cypress: ^13.7.0

## External Services
- Anthropic Claude API (API key required, stored via firebase functions:secrets:set)

## Browser Requirements
- Modern browsers with ES6+ support
- JavaScript enabled
- Cookies enabled (for authentication)

## Recommended IDE Extensions
- ESLint
- Firebase
- Tailwind CSS IntelliSense

## System Requirements
- Minimum 4GB RAM for development
- 500MB free disk space
- Internet connection for Firebase services
